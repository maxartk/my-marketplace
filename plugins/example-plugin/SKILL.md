# Example Plugin

Цей плагін — шаблон для створення власних плагінів.

## Що він робить

Опис того, яку задачу виконує цей плагін.

## Як використовувати

Команда: `скажи "текст"`

## Створення власного плагіну

1. Скопіюй цю папку
2. Зміни назву папки та `manifest.json`
3. Напиши свою інструкцію в `SKILL.md`
4. Залей на GitHub
